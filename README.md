# Watson Statusbar

A simple GUI wrapper around [watson](https://github.com/TailorDev/Watson) using Python and [rumps](https://github.com/jaredks/rumps).


## Prerequisites

- python 3.8
- [watson](https://github.com/TailorDev/Watson)


## Installation

```bash
$ make install
```


## Running the app

```bash
$ make run
```
